Algebra 2 – Graphs & Function Behavior

This guide focuses on understanding how functions behave, how graphs are shaped, and how algebra connects to visual intuition.

1) Functions as Machines

A function takes an input and produces exactly one output.

$f(x) = x^2$
Input: $-2$ → Output: $4$

Graphs show all possible inputs and outputs at once.

2) Key Graph Features

  • x-intercepts (zeros): where $f(x)=0$
  • y-intercept: $f(0)$
  • Domain: allowed x-values
  • Range: possible outputs
If $f(x) = x^2 - 4$, the zeros are $x = \pm 2$.

3) Polynomials and Their Shape

The degree of a polynomial controls its overall behavior.

Degree Name Max Turns
1 Linear 0
2 Quadratic 1
3 Cubic 2
n Polynomial n − 1

4) End Behavior

End behavior describes what happens to a graph as $x \to \pm \infty$.

  • Even degree + positive coefficient → both ends up
  • Even degree + negative coefficient → both ends down
  • Odd degree → ends go opposite directions
$f(x) = x^4 - 3x^2$ → both ends up
$g(x) = -x^3 + 2$ → left up, right down

5) Zeros and Multiplicity

The number of times a factor repeats changes how the graph crosses the x-axis.

  • Odd multiplicity → graph crosses
  • Even multiplicity → graph touches and turns
$(x-2)^2$ → touches at $x=2$
$(x+1)^3$ → crosses at $x=-1$

6) Function Transformations

Starting with a base function like $y=x^2$:

  • $y = x^2 + 3$ → up
  • $y = (x-2)^2$ → right
  • $y = -x^2$ → reflect over x-axis
  • $y = 2x^2$ → vertical stretch

7) Exponential vs Polynomial Growth

Polynomial Exponential
$x^2$ $2^x$
Grows steadily Explodes quickly
Additive thinking Multiplicative thinking

8) Rational Functions & Asymptotes

  • Vertical asymptote: denominator = 0
  • Horizontal asymptote: compare degrees
$f(x) = \frac{1}{x-2}$
Vertical asymptote at $x=2$

9) Why This Matters

Algebra 2 is about recognizing patterns:

  • Predicting graphs without plotting every point
  • Understanding growth and change
  • Preparing for calculus and real-world modeling

Use this guide alongside practice problems and graphing tools. Understanding behavior beats memorization.
