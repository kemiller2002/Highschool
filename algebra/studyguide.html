Algebra Mini Study Guide + Practice Set

Topics: imaginary numbers, exponents/roots, factoring, equation types, inverses, radical equations, extraneous solutions.

1) Imaginary Numbers (i)

Definition: $i=\sqrt{-1}$, so $i^2=-1$.

Powers of $i$ (cycle of 4)

  • $i^1=i$
  • $i^2=-1$
  • $i^3=-i$
  • $i^4=1$

Tip: Reduce powers mod 4. Example: $i^{18}=i^2=-1$.

Multiplication example: $(3i)(2i)=6i^2=-6$.

2) Exponents & Roots

Fractional exponents: $$a^{m/n}=\sqrt[n]{a^m}$$

Example: $$11^{4/5}=\sqrt[5]{11^4}$$

Root identity: $$\sqrt{x^2}=|x|$$

3) Factoring Polynomials

Difference of cubes: $$a^3-b^3=(a-b)(a^2+ab+b^2)$$

Example: $$x^3-27=(x-3)(x^2+3x+9)$$

Difference of squares: $$a^2-b^2=(a-b)(a+b)$$

5) Inverse Functions

  1. Write $y=f(x)$
  2. Swap $x$ and $y$
  3. Solve for $y$
  4. Rename as $f^{-1}(x)$

Example: if $f(x)=\frac{1}{2}x-3$, then $f^{-1}(x)=2x+6$.

6) Radical Equations + Extraneous Solutions

Method: isolate the root → square → solve → check.

Squaring can create extraneous solutions, so always plug back in.
